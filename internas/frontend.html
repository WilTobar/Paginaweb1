ANGULAR

Descripción: Angular es un framework mantenido por Google que desarrolla aplicaciones web en TypeScript que utiliza el modelo vista controlador; siendo eficiente y escalable. Es una buena opción para el desarrollo de aplicaciones web complejas que requieran de una gran cantidad de iteración y funcionalidades avanzadas

Características:

  1. Su arquitectura se basa en componentes
  2. Utiliza el lenguaje de programación TypeScript
  3. Enfoque en la experiencia del usuario
  4. Gestión eficiente de las dependencias entre componentes
  5. Dispone e gran cantidad de recursos, herramientas y soluciones disponibles para desarrolladores
